# Ayini Home Products — Backend API

Node.js + Express + SQLite backend for Ayini Home Products e-commerce.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Create environment file
cp .env.example .env
# Fill in your values in .env

# 3. Start development server
npm run dev

# 4. API is live at
http://localhost:5000/api/health
```
